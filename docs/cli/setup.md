# `aeonsage setup`

Initialize `~/.aeonsage/aeonsage.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
aeonsage setup
aeonsage setup --workspace ~/aeonsage
```

To run the wizard via setup:

```bash
aeonsage setup --wizard
```
