# `aeonsage tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
aeonsage tui
aeonsage tui --url ws://127.0.0.1:18789 --token <token>
aeonsage tui --session main --deliver
```
