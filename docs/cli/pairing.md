# `aeonsage pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
aeonsage pairing list whatsapp
aeonsage pairing approve whatsapp <code> --notify
```
