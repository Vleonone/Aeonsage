## General

### Q1: What is AeonSage?

AeonSage is a **Sovereign Intelligence Operating System** — a complete infrastructure for deploying, managing, and scaling autonomous AI agents across enterprise environments.

Unlike simple chatbots, AeonSage provides:
- Multi-agent orchestration with CEO-worker delegation
- Enterprise-grade security and compliance
- Unified management across 10+ communication channels
- Real-time Workflow Canvas for visual supervision

---

### Q2: Is AeonSage open source?

**Yes.** AeonSage Open Source is released under the MIT license and available on GitHub.

We also offer **AeonSage Pro** with additional enterprise features:
- Multi-Worker Parallel Mode
- Workflow Canvas visualization
- Advanced security (VDID identity)
- Managed cloud deployment

---

### Q3: What are the system requirements?

**Minimum**:

**Recommended for Production**:
- 4GB+ RAM

### Q5: How does the Multi-Worker mode work?

**Pro Feature**: AeonSage Pro allows your CEO Agent to autonomously spawn multiple Worker Agents to complete tasks in parallel.

Example:
- Without Pro: Tasks execute sequentially (25 minutes)
- With Pro: 3 Workers execute in parallel (5 minutes)

The Workflow Canvas provides real-time visualization of all active workers.

---

### Q6: What is VDID?

**VDID** (Verifiable Decentralized Identity) is our identity protocol for AI agents:

- Each agent has a cryptographic identity
- All actions are signed and auditable
- Supports on-chain reputation tracking
- Enables multi-agent trust relationships

Available in Pro Enterprise tier.

## Security

### Q7: How is my data protected?

AeonSage follows **Sovereign-First** principles:

- **Self-hosted**: Your data never leaves your infrastructure
- **End-to-end encryption**: All communications encrypted
- **God Key**: Emergency kill switch for all agents
- **Safety Gates**: Granular permission controls

**Pro Cloud** adds:
- Managed SSL/TLS
- DDoS protection
- Automatic security updates
- SOC2 compliance

---

### Q8: Can I use my own API keys?

Yes. AeonSage supports all major AI providers:
- OpenAI (GPT-4, GPT-3.5)
- Anthropic (Claude)
- Google (Gemini)
- Local models (Ollama)

Your API keys are stored encrypted locally.

## Deployment

### Q11: Should I use self-hosted or cloud?

**Self-hosted (Open Source)**:
- ✅ Full control
- ✅ No ongoing costs
- ⚠️ You manage security, backups, updates

**Pro Cloud**:
- ✅ Managed security and updates
- ✅ Automatic backups
- ✅ DDoS protection
- ✅ 24/7 monitoring

**Recommendation**: Use Pro Cloud for production. Use self-hosted for development.

---

### Q12: How do I upgrade from Open Source to Pro?

1. Sign up at [pro.aeonsage.org](https://pro.aeonsage.org)
2. Choose your tier (Personal or Enterprise)
3. One-click deploy to cloud
4. Import your existing configuration

Your data and settings transfer automatically.
